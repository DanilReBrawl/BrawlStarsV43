# Как играть в BS DRB:
1. скачать сервер и apk
2. установить апк
3. скачать pydroid (если хотите запустить с телефона)
4. открыть в pydroid Core.py, расположенном в папке сервера
5. нажмите на кнопку запуска
6. теперь открывай игру и играй
